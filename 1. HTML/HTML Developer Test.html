HTML Developer Test

Instructions:

  • This is a timed test. You have 60 minutes to complete all questions.
  • Answer each question in the space provided.
  • Make sure to write clean and well-commented code.

Section 1: Multiple Choice Questions (MCQs)

  1. Which HTML tag is used to display the largest heading?
    • a) <heading>
    • b) <h6>
    • c) <h1>
    • d) <head>
  2. Which of the following is the correct HTML tag for inserting an image?
    • a) <image>
    • b) <src>
    • c) <img>
    • d) <picture>
  3. Which attribute is used to specify a URL in an anchor tag?
    • a) link
    • b) url
    • c) href
    • d) src
  4. Which tag is used to define a block of text as a quote?
    • a) <block>
    • b) <blockquote>
    • c) <q>
    • d) <quote>
  5. What is the correct way to embed an external stylesheet in HTML?
    • a) <style src="style.css">
    • b) <link rel="stylesheet" href="style.css">
    • c) <css src="style.css">
    • d) <style type="stylesheet">
  6. Which HTML tag is used to create an ordered list?
    • a) <ul>
    • b) <ol>
    • c) <list>
    • d) <order>
  7. Which tag is used to create a form in HTML?
    • a) <input>
    • b) <form>
    • c) <field>
    • d) <textarea>
  8. What is the purpose of the <meta> tag in HTML?
    • a) It adds a link to the header.
    • b) It defines the character set used in the document.
    • c) It provides metadata such as author and description.
    • d) Both b and c.
  9. Which tag is used for inserting an audio file into a web page?
    • a) <audio>
    • b) <media>
    • c) <video>
    • d) <sound>
  10. Which HTML tag is used to define a navigation section?
    • a) <nav>
    • b) <navigate>
    • c) <section>
    • d) <menu>

Section 2: Short Answer Questions

  1. Explain the difference between the <b> and <strong> tags. Provide an example of each.
  2. Describe the purpose of the <iframe> tag. How would you use it to embed a YouTube video in your HTML page?
  3. What is the difference between <ol> and <ul> in HTML?
  4. What does the alt attribute in the <img> tag do? Why is it important for web accessibility?
  5. What is the role of the <form> element in HTML? List at least 3 input elements commonly used inside forms.

Section 3: Theoretical Questions

  1. What is the difference between the <div> and <section> tags in HTML?
  2. What are semantic HTML elements, and why are they important?
  3. Explain the concept of the <header> and <footer> elements in HTML5.
  4. What is the significance of the <noscript> tag in HTML?
  5. What is the difference between a block-level element and an inline element in HTML? Provide examples of each.

Section 4: Advanced HTML Tags & Concepts

  1. Explain and demonstrate how you would use the following HTML tags:
    • <mark>
    • <sub>
    • <sup>
    • <code>
    • <kbd>
  2. Write an HTML code snippet that uses a semantic tag like <article>, <header>, and <footer>. Your page should also contain at least one image, one heading, and a paragraph of text.
  3. Describe how you would use the <audio> and <video> tags with their source elements to display a video and an audio file on a page. Include controls for both media types.
